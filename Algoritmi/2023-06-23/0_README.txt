CONTENUTO DI QUESTA CARTELLA

- il .csv file coi voti per questo appello
- il file Spiegazioni.md spiega come puoi gestire i voti nel tuo portafoglio,
  come acquisiti per il modulo di Algoritmi o per quello di Complessità, come migliorare la tua valutazione o richiedere la registrazione di un voto per l'intero insegnamento.
- il folder contenente tutte le vostre sottomissioni durante l'appello.

Puoi testare o rivedere le tue soluzioni sottoponendole al server TALight dedicato. Potrai così prendere le tue verifiche e/o completare la tua esperienza d'esame.


Per simulare appelli di esame anteriori a 2023-02-20 usa invece il CMS algo-simula-prove:

https://rizzi.olinfo.it/algo-simula-prove

cui potrai accedere con le tue credenziali (se non ne hai o non le ricordi puoi crearne di nuove, ma non accetta la creazione di username già registrate)


Se vuoi accedere ai vari sorgenti del pacchetto problema (soluzioni, testo, generatori di istanze, ed eventuali grader e/o correttori e verificatori), puoi visionare tutto da:
 https://github.com/romeorizzi/esami-algo-public/
